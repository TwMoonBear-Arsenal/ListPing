# ListPing

依給定主機清單偵測是否上線
使用 -h 獲取使用說明

> py.exe .\listping.py -h
> 
> usage: listping.py [-h] input_file_path
> 
> 依給定主機清單偵測是否上線
> 
> positional arguments:
>   input_file_path  主機清單檔案位置
> 
> optional arguments:
>  -h, --help       show this help message and exit
>
> example usage: py.exe .\listping.py example_hosts_list.txt
